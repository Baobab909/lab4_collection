# Лабораторная работа №4: Симуляция библиотеки с пользовательскими коллекциями

## Описание проекта

Лабораторная работа по реализации системы управления библиотекой с использованием пользовательских коллекций и псевдослучайной симуляции. Проект включает:
- Две пользовательские коллекции (списковая и словарная)
- Модель предметной области (книги, библиотека)
- Псевдослучайную симуляцию событий библиотеки
- Поддержку магических методов Python
- Систему индексации и поиска книг

---

## Запуск

```bash
python main.py
```

## Функции и возможности
- Модель книги (Book)
- Списковая коллекция (BookCollection)
- Словарная коллекция (IndexDict)
- Библиотека (Library)
- Симуляция (LibrarySimulation)

## Основные события симуляции
- Добавление новой книги
- Удаление случайной книги
- Поиск по автору/жанру/году
- поиск несуществующей книги
- обновление случайной книги


## Магические методы
* BookCollection
- __getitem__(key) - доступ по индексу и срезу
- __iter__() - итерация по коллекции
- __len__() - количество элементов
- __contains__(item) - проверка наличия
- __repr__() - строковое представление

* IndexDict
- __getitem__(key) - доступ по ключу
- __setitem__(key, value) - установка значения
- __delitem__(key) - удаление по ключу
- __len__() - количество элементов
- __iter__() - итерация по ключам
- __repr__() - строковое представление

* Book
- __repr__() - строковое представление
- __contains__(item) - поиск подстроки в информации о книге
- __eq__(other) - сравнение книг по ISBN

* Library
- __repr__() - строковое представление библиотеки
